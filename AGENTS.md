# Project Rules

## Stack
- Vite + React + TypeScript
- Tailwind CSS
- No external API calls for drill content or progress

## Implementation
- Keep dependencies minimal.
- Persist learner progress in `localStorage`.
- Code comments must be written in English.
- Do not commit `.env`, private keys, tokens, or generated secret files.

## UI
- Mobile first.
- Body text must remain at least 16px.
- Numeric inputs must remain at least 20px.
- Tap targets must remain at least 44px.
