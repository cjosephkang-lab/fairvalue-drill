# フェアバリュードリル

M&Aバリュエーション学習用のスマホ優先SPAです。EV/EBITDA、株式価値、DCF、フェアネス判定の基礎を毎朝短時間で練習できます。

## セットアップ

```bash
npm install
npm run dev
```

進捗は使っているブラウザの `localStorage` にだけ保存されます。ログイン、サーバー、共有データベースはありません。

## ビルド

```bash
npm run build
```

## GitHub Pages デプロイ

この設定は GitHub リポジトリ名が `fairvalue-drill` である前提です。

```bash
npm run deploy
```
